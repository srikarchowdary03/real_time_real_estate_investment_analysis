calculateMonthlyPayment(principal, annualRate, years) → {number}

Calculate monthly mortgage payment using PMT formula Uses the standard amortization formula to calculate monthly payment: Payment = P * [r(1+r)^n] / [(1+r)^n - 1] Where: P = principal, r = monthly rate, n = number of payments
Parameters:
Name Type Description
principal number Loan principal amount in dollars
annualRate number Annual interest rate as percentage (e.g., 7.0 for 7%)
years number Loan term in years (e.g., 30)
Source:
  • utils/investmentCalculations.js, line 29
Returns:
Monthly payment amount in dollars
Type
number
Example
// Calculate payment for $200,000 loan at 7% for 30 years
const payment = calculateMonthlyPayment(200000, 7.0, 30);
console.log(payment); // 1330.60

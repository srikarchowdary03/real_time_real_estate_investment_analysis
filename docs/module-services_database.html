Module: services/database

Complete database service for managing saved properties in Firestore. Provides CRUD operations for user's saved property portfolio including saving, retrieving, updating, and deleting properties. Handles property data, rent estimates, analysis results, user notes, and tags. Document ID Pattern: {userId}_{propertyId} Collection: savedProperties Used by: PropertyCard, ExpandedPropertyView, PropertyAnalysisPage, MyProperties, useSavedProperties hook, and PropertyDetails components.

Module: contexts/AuthContext

(static, constant) AuthProvider

Authentication Provider Component Wraps the application to provide authentication context to all child components. Listens to Firebase authentication state changes and updates context automatically. Displays children only after authentication state is loaded.

(static, constant) useAuth

Custom authentication hook Provides access to authentication context. Must be used within AuthProvider. Returns current user and loading state from Firebase authentication.
Source:
  • contexts/AuthContext.jsx, line 77
Examples
function MyComponent() {
  const { currentUser, loading } = useAuth();
  
  if (loading) {
    return <div>Loading...</div>;
  }
  
  if (!currentUser) {
    return <div>Please sign in</div>;
  }
  
  return <div>Welcome {currentUser.displayName}!</div>;
}
// Using in property operations
function SavePropertyButton({ property }) {
  const { currentUser } = useAuth();
  
  const handleSave = async () => {
    if (!currentUser) {
      alert('Please sign in to save properties');
      return;
    }
    await saveProperty(currentUser.uid, property);
  };
  
  return <button onClick={handleSave}>Save</button>;
}
